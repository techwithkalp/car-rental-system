# 🚗 Car Rental System (PHP & MySQL)

## 📌 Overview

This is a complete **Car Rental System** built using PHP and MySQL.\
It allows users to browse cars, book rentals, and manage bookings, while
the admin can control cars, users, and bookings from the dashboard.

------------------------------------------------------------------------

## ✨ Features

### 👤 User Side

-   User registration & login\
-   Browse available cars\
-   View car details\
-   Book a car\
-   Cancel bookings\
-   Booking history\
-   Auto return system

### 🔑 Admin Panel

-   Admin login system\
-   Dashboard overview\
-   Add / Edit / Delete cars\
-   Manage users\
-   Manage bookings\
-   Update booking status\
-   View feedback\
-   Admin profile management

## ⚙️ Installation Guide

1.  Clone the repository: git clone
    https://github.com/techwithkalp/car-rental-system.git

2.  Move project to:

    -   XAMPP → htdocs
    -   WAMP → www

3.  Import Database:

    -   Open phpMyAdmin
    -   Create a new database (car_rental)
    -   Import carrental.sql

4.  Configure Database: Open config.php and update: \$conn =
    mysqli_connect("localhost", "root", "","car_rental");

5.  Run Project: http://localhost/car_rental_system
